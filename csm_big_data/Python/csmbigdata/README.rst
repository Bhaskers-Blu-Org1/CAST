CSM Big Data Interface Packages
===============================

    :maxdepth: 3
    :caption: Table of Contents
    :name: csmbigdatatoc

The `csmbigdata` packages provide a number of modules for interfacing with the Big Data Store
and CSM APIs. For more granular details, please consult the README documentation in each 
sub directory.

:config: A collection of modules that define a standardized manner in which to take advantage 
    of the `configparser` module. This package is optional to the use of the `utils` package.

:utils: A collection of modules that define methods of accessing the Big Data Store and 
    CSM APIs. A helper module is provided for simple `Log Analysis` queries (e.g. getting the 
    start and end time of a job).
